# ConsultarCEPCorreios
Função para consultar nos Correios via CEP

#### Uso:

`ConsultarCEP(90010191)`

#### Retorno:
```
array(4) {                                            
  ["uf"]=>                                                  
    string(2) "RS"                                            
  ["cidade"]=>                                              
    string(12) "Porto Alegre"                                 
  ["bairro"]=>                                              
    string(17) "Centro Histórico"                             
  ["logradouro"]=>                                          
    string(42) "Rua Sete de Setembro - de 1000/1001 ao fim"   
}                                                            
```
